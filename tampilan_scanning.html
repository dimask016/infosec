💡 Rekomendasi Keamanan & End-of-Life (EOL) Assessment
🔒 Rekomendasi Mitigasi:
  • Cookie Security: Tambahkan Secure dan HttpOnly flag pada cookie NEXT_LOCALE untuk mencegah akses JavaScript dan transmisi tidak aman.
  • Cloudflare Configuration: Pastikan origin server tidak terekspos langsung (blokir akses HTTP/HTTPS ke IP asli). Gunakan Authenticated Origin Pulls.
  • Directory Listing: Pada subdomain dengan title "Index of /" (misalnya di 195.88.211.210 dari laporan sebelumnya, tidak ada di antam.com? Tapi dari data subdomain antam tidak ada index of. Namun tetap periksa /icons/ tidak menampilkan file sensitif.
  • SSL/TLS: Hanya TLSv1.2/1.3 sudah baik. Pertahankan.
  • SPF Record: Sudah strict (-all) – sangat baik cegah spoofing.
  • DNSSEC: Aktifkan DNSSEC untuk mencegah cache poisoning.
  • Response Headers: Hapus header x-powered-by (Next.js, Nuxt) untuk menyembunyikan teknologi.
  • Error Handling: Beberapa subdomain mengembalikan 404/503 – pastikan tidak membocorkan informasi internal.
⚠️ End-of-Life (EOL) & Versi Rentan:
  • Next.js & Nuxt: Tidak ada versi spesifik terdeteksi, namun selalu gunakan versi LTS terbaru. Cek secara rutin.
  • OpenSSH 8.0 (tidak muncul di antam.com, tetapi dari name server smpnegeri13mj sebelumnya). Untuk antam.com, tidak ada service SSH yang terekspos publik (port 22 tidak terbuka).
  • PHP: Tidak terdeteksi pada antam.com, sehingga tidak ada risiko EOL PHP.
  • Cloudflare: Bukan EOL, tetapi pastikan menggunakan WAF rules terbaru.
  • Microsoft Azure & AWS: Pastikan service menggunakan versi runtime yang didukung (misalnya Node.js untuk Next.js).
